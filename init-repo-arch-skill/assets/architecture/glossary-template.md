# Глоссарий

Используй этот файл для терминов, которые:

- непонятны агенту без дополнительного контекста
- пользователь отдельно подсветил как важные
- в системе трактуются не так, как агент ожидал по общему смыслу
- стоит вынести как доменно значимые для дальнейшего анализа
